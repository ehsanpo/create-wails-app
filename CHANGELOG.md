# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of create-wails-app
- Interactive CLI with installer-style UX
- Support for Wails v2 (stable) and v3 (experimental)
- Frontend framework support: React, Vue, Svelte, Solid, Vanilla
- Frontend extras: TypeScript, Tailwind CSS, Router, ESLint + Prettier
- GitHub Actions workflows for CI and release
- App features: System tray, single instance, auto-update, native dialogs, config store, deep linking, startup, clipboard, file watcher
- Data & backend: SQLite, encrypted storage, Supabase integration
- Testing setup: Vitest, Playwright, Go tests
- Automatic Wails CLI detection and installation
- 4-layer architecture for clean separation of concerns
- Comprehensive error handling and user guidance
- Post-install UX with next steps and resources

## [0.1.0] - 2026-01-08

### Added
- Initial project structure
- Core CLI functionality
- Template selection system
- Feature patching system
- Documentation (README, CONTRIBUTING, LICENSE)
